**Introduction**

The personal finance tracker application is intended for a user to easily balance their income and expenses on a graphical user interface. Users can track budgetary goals based on chosen categories with complementary graph displays showing transaction history. The ability to filter transactions also demonstrates the application’s sorting and data management features. This app is for anyone with basic computer literacy and finances to manage.  

 

**Getting Started**

The minimum system requirements include a modern web browser (Chrome, Firefox, or Edge), JavaScript enabled, 4 GB of RAM, and a stable internet connection.  

As of now, the application is accessible through a localhost server on the host machine. The user will need to have the latest version of Django, Python, and Pip installed on their system.  

Use the following commands to install the requirements: 

Install Python. 

``` 

Go to https://www.python.org/downloads/ 

Download the latest version and Install to local machine 

``` 

 
Install Pip.  

This should be installed automatically after installing Python. Verify the verison of pip is installed. 

``` 

pip --version 

``` 

 Install Django. 

``` 

pip install django 

```

### Troubleshooting 

1. The application won’t start. 

+ Verify that the name of the URL is spelled correctly and the internet connection is working. If the issue persists, reload the page and do CTRL + R to force reset the page. Whatever dependency is missing, should be reloaded. 

2. There isn’t any data on the Reports page. 

+ Verify that you have transactions created on the Transactions page. The data will only populate if you have entries within the past weeks, months, or years. 

3. I get an error with the entry type when selecting a category. 

+ When a category is created it is associated with an entry type (income or expense). When using the category from the dropdown menu, the transaction entry type must match the category entry type. For example, you cannot have a category called “Food” as an “Expense”, entered in the transactions form as an “Income”. 

  

### Frequently Asked Questions (FAQs) 

1. Is there an amount limit for the transactions? 

+ No, you can enter and track as many quantitative amounts as you’d like. 

2. Can I edit/add transactions to another user’s account? 

+ No, your account is only accessible to you. You cannot change the valeus in another user’s account. 

3. Can I use this app on my phone? 

+ Yes, to an extent. Although the application works on mobile divides, it was originally made to be used on a desktop. 

4. Does the application save my data? 

+ Yes! You will sign up for an account to use the application, so the information will persist in your account. 

5. Can I use the application without the internet? 

+ No, you must be connected to the internet for the web app to function.
